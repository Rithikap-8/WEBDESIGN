What is HTML

HTML stands for Hyper Text Markup Language,HTML is the standard markup language for creating Web pages

HTML describes the structure of a Web page,HTML consists of a series of elements

HTML elements tell the browser how to display the content,The element contains meta information about the HTML page

The purpose of a web browser (Chrome, Edge, Firefox, Safari) is to read HTML documents and display them correctly.With the help of HTML only, we can create static web pages.

CSS stands for Cascading Style Sheets CSS describes how HTML elements are to be displayed on screen, paper, or in other media
